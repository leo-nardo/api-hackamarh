# Introduction

This API is a NestJS REST service adapted for the Hackamarh environmental monitoring MVP.

## Features

- [x] PostgreSQL persistence via TypeORM.
- [x] PostGIS-ready infrastructure for geospatial mission polygons and evidence points.
- [x] JWT authentication, users, roles, and statuses.
- [x] File uploads with local and Amazon S3 drivers.
- [x] Swagger documentation.
- [x] Docker and CI.
